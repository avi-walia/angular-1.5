Inscription

Dans quelle mesure les informations sur mon compte sont-elles protégées sur InfoClientèle Assante?

Les informations de votre compte sont sécurisées grâce au système de codage connu sous le nom de cryptage 128 bits. Le cryptage permet de brouiller toutes les informations entre votre ordinateur et nos serveurs Web pour assurer un niveau élevé de sécurité, le respect de la vie privée et de la confidentialité. Le cryptage 128 bits est actuellement ce qui se fait de mieux en matière de sécurité de l’information.

Qu'est-ce qu'un code d'utilisateur?

Votre code d’utilisateur est un identificateur personnel utilisé par Assante pour vous garantir un accès en ligne sécurisé à votre ou vos comptes.

Remarque : Votre code d'utilisateur est différent de votre numéro de compte Assante et vous permet d'avoir accès à tous vos comptes.

Où puis-je trouver mon code d'utilisateur?

Pour la plupart des clients, votre code d’utilisateur se trouve sur les relevés ou les confirmations d’opération qui vous ont été envoyées par la poste. Vous pouvez également utiliser l’option Récupérer le code d’utilisateur sur la page de connexion.

Si vous avez besoin d'aide, veuillez communiquer avec le Centre de service Assante et un de nos représentants du service financier se fera un plaisir de vous aider.

Remarque : Votre code d'utilisateur est différent de votre numéro de compte. Il vous permet d’avoir accès à tous vos comptes.

Où puis-je trouver mon numéro de compte Assante?

Votre numéro de compte Assante se trouve sur les relevés ou les confirmations d’opération qui vous ont été envoyés par la poste. Si vous n’arrivez pas à trouver votre numéro de compte, communiquez avec le Centre de service Assante et l’un de nos représentants en services financiers se fera un plaisir de vous aider.

Remarque : Vous pouvez utiliser n'importe lequel de vos numéros de compte Assante pour vous inscrire à InfoClientèle Assante.

Pourquoi suis-je obligé de fournir les quatre derniers chiffres de mon numéro d'assurance sociale (NAS) ou numéro d'entreprise (NE) et ma date de naissance?

Votre NAS et date de naissance sont des vérifications de sécurité supplémentaires qui permettront de confirmer votre identité lors de l’inscription.

Aucun numéro d'assurance sociale (NAS), numéro d'entreprise (NE) ou de date de naissance n'est associé à mon compte; comment puis-je m'inscrire à InfoClientèle Assante?

S'il n'y a pas de NAS ou de date de naissance associé à votre compte (p. ex. un compte de société), veuillez communiquer avec le Centre de service Assante et l'un de nos représentants en services financiers se fera un plaisir de vous aider.

Qu'est-ce qu'un pseudonyme et comment puis-je en créer un?

Le pseudonyme vous permet de vous connecter à InfoClientèle Assante avec votre propre nom personnalisé* à la place du code d'utilisateur qui vous a été attribué.

Vous pouvez créer un pseudonyme lorsque vous vous inscrivez pour la première fois en suivant les messages-guide, ou à tout moment en vous rendant à la section ​​Profil d’InfoClientèle Assante.

*Vous pouvez utiliser votre adresse de courriel comme pseudonyme.

Qu’est-ce que la fonction Mémoriser le code d'utilisateur et comment fonctionne-t-elle?

Si vous cochez la boîte Mémoriser le code d'utilisateur, votre navigateur Web se rappellera de votre code d'utilisateur de sorte que vous n'aurez pas à l'entrer à nouveau lors de votre prochaine connexion à InfoClientèle Assante, si vous utilisez le même appareil. Pour des mesures de sécurité, il vous faudra toujours entrer votre mot de passe.

Si vous avez des difficultés avec la fonction Mémoriser le code d'utilisateur, cliquez ici pour obtenir de l'aide technique.

Comment la question de sécurité d'InfoClientèle Assante lors de l'ouverture de session fonctionne-t-elle?

Lorsque vous vous connectez à partir d’un ordinateur non enregistré, l’écran Question de sécurité s’affichera et vous devrez répondre correctement à votre question de sécurité pour avoir accès à InfoClientèle Assante.

Vous aurez deux chances de répondre correctement à votre première question de sécurité. Si vous ne parvenez pas à répondre à votre première question de sécurité, vous devrez répondre à l’une des autres questions de sécurité.

Important! Si vous ne parvenez pas à répondre à votre deuxième question de sécurité après deux tentatives, votre compte sera bloqué. Pour débloquer votre compte, communiquez avec le Centre de service Assante et l'un de nos représentants en services financiers se fera un plaisir de vous aider.

J'ai oublié mon mot de passe, que dois-je faire?

Si vous ne pouvez pas vous rappeler de votre mot de passe, vous pouvez le réinitialiser en cliquant sur le lien « Vous avez oublié votre mot de passe? » à la page d'accueil.

J'ai oublié les réponses à mes questions de sécurité, que dois-je faire?

Si vous avez oublié la réponse à une de vos questions de sécurité, ou si vous éprouvez des difficultés à répondre à une question de sécurité en particulier, communiquez avec le Centre de service Assante et l'un de nos représentants en services financiers sera en mesure de vous aider.

Qu'est-ce que l'option d'enregistrement de l'ordinateur et comment est-ce que j'enregistre mon ordinateur?

L'option d'enregistrement de l'ordinateur vous permet de contourner l'écran Question de sécurité, qui vous oblige à répondre à l'une de vos questions de sécurité, lorsque vous vous connectez par l'entremise d'un ordinateur enregistré.

RVous pouvez enregistrer votre ordinateur à partir de l'écran Question de sécurité après la connexion initiale, ou à la section Profil sous l'onglet Sécurité.

Important! Par mesure de sécurité, vous ne devriez PAS enregistrer un ordinateur dans un lieu public, tel qu'un café Internet. Vous devriez seulement enregistrer un ordinateur dont vous êtes le propriétaire.

L’écran Question de sécurité pourrait s’afficher lorsque vous vous connectez à partir d’un ordinateur enregistré si :

  • vous ne parvenez pas à ouvrir une session InfoClientèle Assante lors d’une ou plusieurs tentatives, ou
  • Votre dernière session InfoClientèle Assante a eu lieu il y a au moins 90 jours.

Comment puis-je désenregistrer mon ordinateur?

Vous pouvez désenregister votre ordinateur à partir d’InfoClientèle Assante, à tout moment, à la section Profil sous l’onglet Sécurité.

Informations sur le compte

Les renseignements sur mon compte sont-ils à jour?

Les informations sur votre compte sont mises à jour sur une base régulière. Dans la plupart des cas, les informations reflètent celles du jour ouvrable précédent.

Puis-je effectuer des transactions sur InfoClientèle Assante?

Non. Aucune transaction ou opération ne peut être effectuée sur InfoClientèle Assante. Nous vous suggérons de communiquer avec votre conseiller financier qui pourra vous aider avec votre demande.

Comment puis-je ouvrir un autre compte auprès de Gestion de patrimoine Assante?

Pour ouvrir un autre compte auprès de Gestion de patrimoine Assante, nous vous recommandons de vous entretenir avec votre conseiller financier afin de déterminer les types de comptes et les placements qui vous conviennent le mieux.

Comprendre le remboursement de capital

Le remboursement de capital (RDC) est la portion d'une distribution qui réduit le prix de base rajusté (PBR) des parts du fonds n'est pas imposable, et provient du capital de l'investisseur dans le fonds

Par exemple : Distribution de RDC

Un fonds distribue 1,05 $ par part. Le fonds a gagné 0,90 $ par part en intérêts, dividendes et gains en capital. La différence de 0,15 $ est considérée comme un RDC et réduit le PBR de 0,15 $ par part.

Impact des distributions sur le PBR

Le tableau ci-dessous montre l'impact des distributions sur le PBR lorsqu'elles sont versées en espèce ou réinvesties dans le fonds.

Lorsque la distribution est ... Et elle est contient ... Alors le PBR ...
versées en espèces RDC réduit par le montant de RDC
  aucun RDC demeure inchangé
montant réinvesti dans le fonds RDC réduit par le montant de RDC, et
augmente en fonction du montant investi
  aucun RDC augmente en fonction du montant investi

Par exemple : Impact des distributions sur le PBR

L'investisseur investit un montant de 1 000 $ dans un fonds. Le fonds distribue 10 $, dont 3 $ est du RDC.

Si la distribution est de ... alors le PBR est de ... Calcul
versées en espèces 997,00 $ 1 000 $ – 3 $ RDC
montant réinvesti 1 007,00 $ 1 000 $ – 3 $ RDC + 10 $ de montant réinvesti

Impact des gains (pertes) latents

Le RDC est déduit du PBR, qui soit augmente les gains latents, ou réduit la perte latente.

Implications fiscales lorsque le PBR est égal à zéro

Si le PBR a été réduit à zéro, toute autre distribution sera traitées comme des gains en capital, et imposable pour l'année au cours de laquelle la distribution a lieu.

Effet sur la valeur au marché du fonds

Comme toutes les distributions, une distribution de RDC réduira la valeur au marché du fonds. Au fil du temps, la croissance du fonds provenant des intérêts, des dividendes et des gains en capital peut contrebalancer cette baisse.

Documents

Oú puis-je visualiser mes reçus d'impôt, relevés et confirmations d'opération?

Vous pouvez visualiser vos reçus d’impôt, relevés et confirmations d’opération en vous connectant à InfoClientèle Assante et en cliquant sur l’onglet Documents.

Dans la section Documents, des options vous seront offertes pour accéder à vos reçus d’impôt, relevés et confirmations d’opération pour votre ou vos comptes.

Remarque : Si vous ne savez pas si vous devriez avoir un reçu d'impôt pour votre compte, communiquez avec le Centre de service Assante et l'un de nos représentants en services financiers se fera un plaisir de vous aider.

Comment puis-je modifier mes préférences d'envoi de documents?

Vous pouvez configurer vos préférences d’envoi de documents lors de votre inscription, ou à tout moment à la section Profil sous Solution verte.

Important! Si vous choisissez de recevoir des documents électroniques, des exemplaires papiers ne vous seront pas envoyés par la poste. Un avis par courrier électronique vous sera envoyé lorsque les documents sont disponibles en ligne.

Soutien technique

Quels navigateurs Web sont suggérés pour InfoClientèle Assante?

Nous recommandons les navigateurs suivants :

  • Microsoft Internet Explorer
  • Chrome
  • Firefox, ou
  • Safari

Si vous utilisez un autre navigateur Web, il se peut que vous éprouviez des difficultés à accéder à certaines parties d'InfoClientèle Asssante.

Les images, les textes ou les menus ne s'affichent pas correctement, que dois-je faire?

Cela peut être en raison d'un problème de compatibilité avec la version de votre navigateur Web actuel, par exemple, Internet Explorer v11.0. Vous pouvez corriger ce problème en activant le mode compatibilité du navigateur Web.

Si vous éprouvez toujours des difficultés, référez-vous à la section d'aide de votre navigateur Web.

J'éprouve des difficultés à utiliser la fonction Mémoriser votre code d'utilisateur, que dois-je faire?

Pour que vous puissiez utiliser la fonction Mémoriser le code d'utilisateur, votre navigateur doit être en mesure d’accepter les témoins. Si vous avez besoin d'aide, référez-vous à la fonction Aide de votre navigateur.

Qu'est-ce qu'un fichier PDF et de quel logiciel ai-je besoin pour lire un fichier PDF?

Un format de document portable (PDF) est une méthode d'affichage, de sauvegarde et d'impression de documents. Pour ouvrir, sauvegarder et imprimer un fichier PDF, le logiciel Adobe Acrobat Reader ou un autre lecteur de PDF doit être installé sur votre ordinateur.

Cliquez ici pour télécharger votre copie GRATUITE d'Adobe Acrobat Reader.

Puis-je avoir accès à InfoClientèle Assante avec un téléphone intelligent?

Oui. Vous pouvez avoir accès à une version mobile d’InfoClientèle Assante en utilisant votre téléphone intelligent.

J'ai des difficultés à me connecter de nouveau à InfoClientèle Assante. Que dois-je faire?

Il se pourrait que vous ayez fermé la fenêtre du navigateur avant de vous déconnecter d’InfoClientèle Assante. Si cela se produit, fermez toutes les fenêtres du navigateur Web et ouvrez une nouvelle fenêtre de navigateur Web.

Si vous éprouvez toujours des difficultés à vous connecter à InfoClientèle Assante, communiquez avec le Centre de service Assante, et l'un de nos représentants en services financiers se fera un plaisir de vous aider.
